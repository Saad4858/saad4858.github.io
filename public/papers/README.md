# Papers

Drop your paper PDFs in this folder. The publications page references these filenames:

- `kissan-dost.pdf` — Kissan-Dost: Bridging the Last Mile in Smallholder Precision Agriculture with Conversational IoT
- `iot-llm-poster.pdf` — Poster: Bridging IoT Gaps in Developing Regions with LLMs

These are served at `/papers/<filename>` and loaded into the PDF lightbox viewer.

To change which file a paper points to, edit `pdfPath` in `src/data/publications.ts`.
